# README #

### SETUP ###

* npm run setup

### BEFORE RUN SCRIPT ###

* Create two folders: testData , result
* place .csv file in ./testData folder
* msisdn is a REQUIRED param in .csv file. Other param are optional.
* .csv file is case sensitive. Example:

msisdn,telco

60123456789,CELCOM

601987654321,DIGI

### RUN SCRIPT ###

* node read yourCsvFileName
* npm run scriptName
